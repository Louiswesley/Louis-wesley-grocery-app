# Grocery App

This repository now contains a Python Streamlit grocery shopping application.

## Run the App

1. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Start the app:
   ```bash
   streamlit run app.py
   ```

## Notes

- The app provides a shopping interface with cart, checkout, order history, and an optional AI assistant.
- The AI assistant requires a valid Groq API key to function.
